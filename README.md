# Media Search App

A React application for searching Photos, Videos and GIFs.

## Features

- Search Photos
- Search Videos
- Search GIFs
- Responsive Design
- Redux Toolkit State Management

## Tech Stack

- React
- Redux Toolkit
- Tailwind CSS
- Axios
- Vite

## APIs Used

- Unsplash API
- Pexels API
- Giphy API

## Installation

npm install
npm run dev
